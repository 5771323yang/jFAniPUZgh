# 小型餐馆管理系统 java663

## 项目概述

小型餐馆管理系统是基于 Spring Boot 框架开发的，旨在为餐馆提供便捷、高效的管理功能。该系统涵盖了餐馆日常运营的核心功能，支持源码和文档提供。

## 技术栈

### 前端

- HTML
- CSS
- JavaScript

### 后端

- Spring Boot

### 构建工具

- Maven

### 数据库

- MySQL

### 编程语言

- Java

## 功能模块

- 订单管理
- 菜品管理
- 员工管理
- 营业统计

## 系统角色

- 系统管理员
- 餐馆经理
- 员工

## 运行环境

- Java 1.8 或以上版本
- MySQL 5.7 或以上版本

## 部署步骤

1. 环境准备：确保 Java 和 MySQL 已经正确安装
2. 数据库搭建：根据提供的 SQL 脚本创建数据库和表结构
3. 代码编译：使用 Maven 对项目进行编译
4. 配置修改：根据实际环境修改配置文件
5. 应用部署：将编译后的代码部署到 Java 应用的服务器上
